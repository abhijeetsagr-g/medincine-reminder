# Medicine Reminder App

## Requirement

### Core Features
- Save details of medicine. (Name, Dose, Time)
- Save list of medicines.
- Proper State Management
- Save data locally (Using Hive)
- The app must trigger a Notification or Alarm at the scheduled time.

### UX/UI Features
- Show a list of medicines with details.
- The list should be sorted by time.
- Show a placeholder text if the list is empty.
- A simple form with a Time Picker to schedule the medicine.
